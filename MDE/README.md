# Downlevel devices
## Windows Server 2008 R2 SP1, 2012 R2 and 2016
		

Upload the MMASetup-AMD64.exe to Syncro and apply it in the script to the Syncro Bin directory. 

`C:\ProgramData\Syncro\bin\MMASetup-AMD64.exe`
